BODY: A tag BODY é uma das principais tags HTML e é usada para definir o corpo ou conteúdo principal de um documento HTML. Ela é usada para marcar tudo o que deve ser exibido no navegador da web, como texto, imagens, vídeos, links, formulários e outros elementos de conteúdo.

P: A tag P é uma das tags HTML mais comuns e é usada para definir parágrafos de texto em um documento HTML. Essa tag é usada para separar blocos de texto em unidades coerentes e identificáveis, tornando o conteúdo mais legível e fácil de entender.

PRE: A tag PRE é uma tag HTML que é usada para exibir texto pré-formatado na página da web. Essa tag é usada quando é importante manter a formatação exata do texto, como em blocos de código, mensagens de erro ou qualquer outro texto que precise manter a formatação exata, incluindo espaços em branco, quebras de linha e outros caracteres especiais.

H1...H6: As tags H1 até H6 são usadas para definir títulos e subtítulos em um documento HTML. Cada tag representa um nível diferente de hierarquia de título, sendo que H1 é o título principal e H6 é o título de menor importância.

SPAN: A tag SPAN é uma tag HTML que é usada para aplicar estilos a partes específicas de um texto ou para agrupar elementos inline juntos. Ela é usada principalmente para fins de formatação e estilização do conteúdo, como alterar a cor do texto, adicionar um sublinhado, ou destacar parte do texto.

DIV: A tag

é uma tag HTML que é usada para criar uma seção genérica ou um contêiner para agrupar elementos HTML em conjunto. Essa tag é usada principalmente para fins de organização e estruturação de conteúdo na página da web.

HTML: A tag é usada para definir o início e o fim de um documento HTML. Ela é uma das tags mais importantes em uma página da web, pois indica ao navegador que o documento é um arquivo HTML e que deve ser interpretado como tal.

Seletores CSS

SELETOR POR TAG: O seletor por tag em CSS é usado para selecionar todos os elementos HTML que correspondem a uma determinada tag. Por exemplo, o seletor "p" é usado para selecionar todos os elementos P do documento.

SELETOR POR CLASSE: O seletor por classe em CSS é usado para selecionar elementos HTML que possuem uma determinada classe definida em seu código. A classe é um atributo HTML que pode ser adicionado a qualquer elemento para definir um estilo ou comportamento específico para esse elemento.

SELETOR POR ID: O seletor por ID em CSS é usado para selecionar um elemento HTML específico que possui um ID exclusivo definido em seu código. O ID é um atributo HTML que pode ser adicionado a qualquer elemento para identificá-lo exclusivamente em uma página da web.
